    파이썬 Python

    세상의 중심에서 파이썬을 외치다, 2편

    Windows 환경에서 파이썬을 사용하는 방법에 대해 알아봅시다.

    Info

    본 게시글은 파이썬 프로그래밍 언어를 공부하면서 복습하기 위해 강좌 형식으로 작성하였습니다. 다소 장난스럽게 작성되어 있으니 양해바랍니다.

    잘못된 정보나 오탈자 발견 시 댓글로 알려주세요!

    개요

    그림을 그리려면 연필, 붓 등이 필요하고 연주를 하려면 기타, 피아노와 같은 악기가 필요합니다. 프로그래밍 언어도 마찬가지입니다. 프로그래밍 언어를 사용하려면 해당 프로그래밍 언어를 사용할 수 있도록 도와주는 통합개발환경Integrated Development Environment (1)의 설치가 필요합니다.

    1. 코딩, 디버그, 컴파일, 배포 등 프로그램 개발에 관련된 모든 작업을 하나의 프로그램 안에서 처리하는 환경을 제공하는 소프트웨어.

    요즘에는 개발 환경이 많이 좋아져서 대규모의 프로젝트가 아니면 클라우드(1) 방식을 사용하기도 하는데요, 클라우드 방식은 데이터 분석을 배울 때 활용해보도록 하고 지금은 우리가 사용하는 PC에 설치하여 사용하는 방법에 대해 알아보도록 합시다.

    1. 사용자 자신의 컴퓨터에 직접적으로 소프트웨어를 설치하지 않고 언제 어디서나 인터넷 네트워크를 통해 필요한 서비스를 이용할 수 있도록하는 환경.

    Warning

    본 파이썬 강좌는 Windows 64Bit 환경을 기준으로 하고 있습니다. macOS, Linux 등에 대한 게시글은 포함되지 않습니다.

    Python

    설치하기

    파이썬을 사용하려면 파이썬을 설치해야겠죠?

    파이썬 다운로드 페이지에 접속하신 후, 위 사진 속에 보이는 Download Python 버튼을 찾아 클릭합니다. 게시글 작성일 기준으로 최신 버전은 3.12.4이며, 이보다 더 높아도 상관 없습니다.

    다운로드하신 설치 파일을 실행하신 후, Add python.exe to PATH (1) 체크박스에 체크하신 후 Install Now 버튼을 클릭하여 설치를 진행합니다. 설치는 대략 5분 안에 끝납니다.

    1. Add python.exe to PATH 옵션은 환경 변수에 파이썬을 추가하여 언제 어디서든 파이썬을 실행할 수 있도록 합니다.

    시작 메뉴에서 Python 3을 검색했을 때 위 사진과 같이 나타난다면 설치 성공입니다.

    IDE

    파이썬을 설치하면 기본적으로 'IDLE'Integrated Development Environment란 실행 프로그램을 제공합니다. 파이썬의 코드를 작성하고 실행할 수 있도록 하는 개발 도구입니다. 기본적인 문법을 배울 때 사용하기 가볍고 빠르다는 장점이 있지만, 추후 파이썬을 통한 개발을 진행할 때 다소 기능이 부족하고 인터페이스가 불친절한 편이라 잘 사용하지 않습니다. 그래서 본 강좌는 JetBrains 사에서 배포 중인 PyCharm을 설치 후 사용합니다.

    Visual Studio Code로 진행 안 하나요?

    입문자 입장에선 Visual Studio Code나 PyCharm 등 어느 것을 사용해도 상관이 없습니다. 별도로 설정할 줄 아신다면 Python과 Visual Studio Code를 결합해서 사용해도 괜찮습니다.

    다만, 본 강좌는 파이썬만을 대상으로 하기 때문에 PyCharm을 기반으로 합니다.

    PyCharm

    PyCharm은(1) JetBrains 사에서 개발한 파이썬 전용 통합 개발 환경입니다. IntelliJ IDEA에 기반을 두고 있으며, 현재 존재하는 파이썬 개발 도구 중 가장 완성도가 높고 기능이 강력합니다. 무료로 사용할 수 있는 커뮤니티 에디션을 별도로 제공 중이기 때문에 대부분의 파이썬 개발자가 애용하는 편입니다.

    1. '파이참'이라 읽고 발음

    PyCharm 페이지에 접속하신 후 아래로 스크롤하시면 PyCharm Community Edition을 다운로드 할 수 있는 버튼이 나타납니다. 버튼을 클릭하여 설치 파일을 다운로드 받아주세요.

    설치 옵션은 위 사진과 같이 선택한 후 설치합니다. 설치 옵션에 대한 설명은 아래 표와 같습니다:

    옵션 설명
    데스크탑 바로가기 생성 바탕화면에 PyCharm Community Edition을 실행할 수 있는 바로가기를 생성합니다.
    PATH 변수 업데이트(재시작 필요) 환경 변수에서 PyCharm을 추가합니다.
    컨텍스트 메뉴 업데이트 폴더(디렉토리)를 마우스 우클릭 시 나타나는 컨텍스트 메뉴에 PyCharm에서 프로젝트로 지정하여 열 수 있도록 합니다.
    연결 생성 파이썬 확장자(.py)를 가진 파일을 PyCharm Community Edition에서 열 수 있도록 합니다.

    프로그램의 설치 완료와 환경 변수의 올바른 적용을 위해 PC를 재부팅합니다.

    PyCharm

    기본 설정

    PyCharm Community Edition을 실행하면 위 사진처럼 약관 동의와 관련된 창이 나타납니다.

    음... 모두 영어로 작성되어 있어서 잘 모르겠네요. 동의하고 계속해줍시다.

    JetBrains 사의 제품 향상을 위해 익명의 데이터를 보낼 지 동의를 구하는 창입니다. 개인적으로는 익명이라 해도 잘 안 믿는 편이라 보내지 않음에 체크하곤 합니다. 본 강좌에선 보내지 않음을 클릭하도록 하겠습니다.

    새 프로젝트

    PyCharm을 실행하면 보이는 창입니다.

    자세히 보시면 새 프로젝트 생성을 위한 New Project 버튼이 존재합니다. 클릭해주세요.

    프로젝트를 설정하는 영역입니다.

    Name과 Location은 각각 프로젝트의 이름과 경로입니다. 이름은 원하는대로 설정해주시고, 경로에는 프로젝트를 저장하고 관리할 폴더(디렉토리)를 설정합니다. 본 강좌는 아래와 같이 프로젝트를 관리합니다.

    📁 C:/
└── 📁 Users/
    └── 📁 Documents/
        └── 📁 PycharmProjects/


    사진에 표시가 안 되어 있지만... Create a welcome script를 체크해 주세요.

    Interpreter Type은 Custom environment를 클릭해주신 후 아래 표와 같이 설정합니다. Python Path는 파이썬이 설치되어 있는 경로를 말합니다.

    Environment Type Python Path
    Select existing Python C:/Users/{your name}/AppData/Local/Programs/Python/Python{버전}/python.exe
    AppData가 안 보이시나요?

    AppData와 같은 폴더는 기본적으로 숨김 처리로 되어있기 때문에, 위 사진 속에 표시된 눈동자 아이콘을 클릭하면 숨겨진 폴더를 표시할 수 있습니다.

    실행

    새 프로젝트 생성할 때 Create a welcome script에 체크하면 자동으로 main.py도 생성됩니다.

    main.py에 정체를 알 수 없는(?) 코드가 작성되어 있는데요, 일단 한 번 실행해보도록 합시다.

    프로그램 최상단에 있는 ▷ 버튼을 누르시거나, Shift+F10 단축 키를 눌러 실행할 수 있습니다. 실행하면 아래와 같은 결과가 나타납니다:

    Hi, PyCharm

    연습문제

    1. Hi, PyCharm 대신 Hi, {내 이름}으로 출력되게 해보세요. { 내 이름}은 진짜 자신의 이름으로 치환합니다.
