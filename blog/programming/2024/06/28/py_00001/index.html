    파이썬 Python

    세상의 중심에서 파이썬을 외치다, 1편

    컴퓨터가 등장하면서 수 많은 프로그래밍 언어가 등장했습니다. 오늘 우리가 배울 파이썬은 그 수 많은 언어 중 하나입니다.

    Info

    본 게시글은 파이썬 프로그래밍 언어를 공부하면서 복습하기 위해 강좌 형식으로 작성하였습니다. 다소 장난스럽게 작성되어 있으니 양해바랍니다.

    잘못된 정보나 오탈자 발견 시 댓글로 알려주세요!

    개요

    1989년 12월, 파이썬의 창시자인 귀도 반 로섬Guido van Rossum은 크리스마스 때 시간을 보내기 위해 프로그래밍 프로젝트를 찾아 진행하였습니다. 그렇게 진행되어 만들어진 것이 바로 오늘 우리가 배울 파이썬Python이며, 1991년에 최초 공개되어 현재까지 다양한 분야에서 많이 활용되고 있습니다.

    귀도 반 로섬은 몬티 파이썬의 플라잉 서커스Monty Python's Flying Circus의 열렬한 팬이었기 때문에 프로젝트의 제목을 파이썬으로 하게 되었다고 합니다.1

    파이썬의 로고를 보시면 좌측 상단의 뱀 모양으로 이루어진 것을 볼 수 있습니다. 파이썬이라는 명칭은 몬티 파이썬의 플라잉 서커스에서 이름을 따오긴 했지만, 파이썬의 사전적 의미를 찾아보면 '비단뱀'이라 나타납니다. 파이썬의 로고와 대부분의 파이썬 관련 서적들이 뱀 모양을 사용하는 이유는 바로 이 때문입니다.

    파이썬을 배워야하는 이유?!

    인기

    TIOBE Index에 따르면 가장 많은 점유율을 가진 프로그래밍 언어는 파이썬으로 나타났습니다. 또한, 위 차트를 보시면 알 수 있듯이 등락폭이 조금 있지만... 매년 그 인기가 꾸준히 우상향 중인 걸 확인할 수 있습니다. 물 들어올 때 노 저으라고 하듯이 파이썬의 인기가 많을 때 배워보도록 합시다.

    간단한 문법

    파이썬의 정신에서 파이썬은 간결함과 가독성을 중요하게 생각하고 있습니다. 때문에 파이썬의 코드 문법은 타 프로그래밍 언어에 비해 쉽고 간결한 편에 속합니다. 그래서 첫 프로그래밍 언어로 선택하여 배우기도 하고, 학교에서 교육용으로 채택해 많이 가르치기도 합니다. 한국의 경우 파이썬보단 스크래치를 사용하는 것 같습니다.

    파이썬의 정신을 준수하며 코드를 작성하는 것을 파이써닉Pythonic이라 합니다. 파이썬스러운 코드라고 해석할 수 있는데요, 단순히 가이드라인을 준수하는 것을 넘어 코드가 복잡하지 않고 그 의미가 명확 · 명백하게 보이도록 작성하는 걸 의미합니다. 파이써닉한 코드를 작성하는 굇수분들(?) 덕분에 깃허브에서 파이썬으로 제작된 프로젝트를 살펴보면 생각보다 코드를 읽고 분석하는 게 어렵지 않습니다!!!

    다양한 분야

    사람마다 잘하는 분야가 있듯이 프로그래밍 언어도 특화된 분야가 존재합니다. 파이썬은 대략 아래와 같은 분야에서 주로 활용되고 있습니다.

    웹 프로그래밍

    Django, Flask와 같은 프레임워크(1)를 사용하여 자신만의 웹 페이지를 만들어 운영할 수 있습니다. Instagram, Pinterest 등이 파이썬과 Django를 통해 만들어진 대표적인 웹 사이트입니다. 파이썬은 흔히 개발 속도가 빨라서 생산성이 높다고 합니다. 짧은 기간 내에 원하는 결과물을 만들어낼 수 있습니다.

    1. 일정한 틀, 규약을 통해 개발할 수 있도록 하는 라이브러리의 묶음집

    데이터 및 통계

    R, Julia 등 데이터 분석과 통계에 특화된 언어가 별도로 있지만, 파이썬에서도 이를 수행할 수 있는 라이브러리(1)가 많이 생겨나면서 대체 언어로 많이 사용하고 있습니다. 분석에 많이 사용하는 대표적인 라이브러리로 numpy, pandas가 있습니다.

    1. 소프트웨어 개발을 위해 제공하는 코드 모음집

    임베디드

    Raspberry Pi와 같은 소형 임베디드 장치를 활용할 때 파이썬을 사용합니다. 파이썬 언어는 특정 플랫폼에 종속되지 않는다는 특징이 있기 때문에 다양한 환경에서 결합하여 사용할 수 있습니다. 소형 임베디드 장치와 파이썬을 활용해 CCTV, 게임기 등 나만의 기기를 만들어볼 수 있습니다. (기기 값은 별도)

    활용하기 힘든 분야

    파이썬은 인터프리터 방식의 언어라는 점에서 태생적 한계가 존재합니다. 즉, 컴파일(1) 방식에 비해 속도가 매우 느리기 때문에 고성능을 요구하는 분야에서 활용하기엔 적합하지 않습니다.

    1. 소스 코드를 한 번에 기계어로 번역한 후 실행 가능한 프로그램을 만들어내는 방식

    게임

    2D 게임은 고성능을 요구하는 경우가 별로 없기 때문에 파이썬을 활용해 만드는 경우가 있습니다. 하지만, 3D 게임의 경우 사용자가 사용하는 PC의 CPU와 GPU를 최대한 활용해 성능을 이끌어내야 하기 때문에 파이썬을 사용하는 건 별로 적합하지 않습니다. 고성능을 요구하기엔 태생적 한계가 존재하고, 섬세하게 하드웨어를 제어하기엔 복잡하기 때문입니다. 때문에 C++ 언어와 파이썬을 결합해 사용하는 경우도 있지만... 이러한 프로젝트는 많이 없습니다. 그럼에도 불구하고 많이 사용하는 라이브러리와 프레임워크로 pygame, Panda3D가 있습니다.

    모바일

    Android, iOS 환경에서 작동하는 앱을 개발하는 데 적합한 도구가 거의 없습니다. Android 환경의 경우 JAVA나 Kotlin을 주로 사용하고, iOS에선 Swift, Objective-C를 주로 사용하고 있어 파이썬을 사용하기엔 그 정보나 도구가 매우 부족합니다. (사실상 불가능한 수준)

    마무리

    파이썬 프로그래밍 언어에 대한 개요와 간단하게 활용되는 분야를 확인해 보았습니다.

    게임 분야가 눈으로 직접 보면서 개발할 수 있어 가장 재밌는(?) 분야이기도 한데, 파이썬에 적합하지 않다고 해서 실망하실 수도 있습니다. 그렇다고 본 강좌에서 생략하는 것은 아니니 걱정마세요. 간단한 2D 게임을 만들어보는 미니 프로젝트가 있습니다. 게임 분야 외에도 다양한 분야를 조금이라도 찍먹할 수 있는 미니 프로젝트도 있습니다.

    프로그래밍 언어도 언어다!

    프로그래밍 언어도 우리가 사용하는 언어와 같습니다. 영어를 배울 때 기본적인 단어를 암기하고, 5형식 문장을 배우고 활용하듯이 프로그래밍 언어도 이와 같습니다. 프로그래밍 언어에서 사용하는 기본적인 용어와 기술을 배워 개쩌는 프로그램을 만들어 보도록 합시다.
